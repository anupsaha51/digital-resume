##  Getting Started

###  Clone or Download

git clone https://github.com/anupsaha51/digital-resume.git
cd digital-resume


### Install Dependencies
npm install

### Start Development Server
npm start

Then open your browser and go to http://localhost:3000 to view your digital resume.
